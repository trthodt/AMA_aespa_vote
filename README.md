# 🌟 Aespa AMA Voting Extension

A Chrome Extension that helps you quickly vote for **aespa** at the American Music Awards (AMA) voting site.

---

## ✨ What it does

* Automatically opens the **Best Female K-pop Artist** category
* Selects **aespa**
* Helps you increase votes faster with minimal manual steps

---

## ▶️ How to use

1. Download and extract the extension `.zip`
2. Open Chrome and go to:

   ```
   chrome://extensions/
   ```
3. Turn on **Developer mode**
4. Click **Load unpacked** and select the extracted folder (aespa_voting/src)
5. Go to the AMA voting website
6. Click the extension icon
7. Click the button to start voting

---

## ⚠️ Note

* Make sure you are logged in before voting
* The extension interacts directly with the website UI
* It may stop working if the website updates its layout
